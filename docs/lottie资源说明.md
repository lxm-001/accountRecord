# Lottie 资源说明

请从以下链接下载水位球动画 JSON 文件，重命名为 `water_ball.json` 放到 `app/src/main/res/raw/` 目录：

推荐资源（免费）：
1. https://lottiefiles.com/animation/water-level
2. https://lottiefiles.com/free-animations/water-loading
3. https://lottiefiles.com/free-animations/liquid-loading
4. https://lottiefiles.com/animation/circular-progress-bar

下载步骤：
1. 打开上述链接，选择一个喜欢的水位球/液体进度动画
2. 点击下载，选择 "Lottie JSON" 格式
3. 将下载的 .json 文件重命名为 `water_ball.json`
4. 放到 `app/src/main/res/raw/` 目录下

代码已经准备好了，放入文件后直接编译运行即可。
