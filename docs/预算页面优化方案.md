# 预算页面优化方案

> 参考随手记、微信记账本、鲨鱼记账、叨叨记账等主流记账 App 的预算模块设计。

---

## 现状问题

当前预算页只有：
- 一个总预算进度条
- 分类预算列表
- 无预算时的引导卡片
- 右下角设置预算 FAB

页面内容单薄，缺少数据洞察和交互深度。

---

## 优化方案

### 1. 顶部月份切换器

和首页、报表保持一致的交互风格：
- 左右箭头切换月份
- 点击日期文字弹出月份选择器
- 切换月份后刷新当月预算数据

### 2. 预算总览卡片（替代当前的纯进度条）

用一个 ElevatedCard 展示核心数据：

```
┌─────────────────────────────────┐
│  本月预算总览                     │
│                                  │
│     ¥3,000.00                    │
│    ──────────── (环形进度)         │
│     已用 68%                     │
│                                  │
│  预算  ¥3,000    已花  ¥2,040    │
│  剩余  ¥960      日均可用 ¥96    │
└─────────────────────────────────┘
```

关键数据：
- 环形进度图（绿/橙/红三色阈值）
- 预算总额、已花金额、剩余金额
- 日均可用额度 = 剩余 ÷ 本月剩余天数
- 预算使用百分比

### 3. 分类预算卡片列表

每个分类预算用 ElevatedCard 展示：

```
┌─────────────────────────────────┐
│  🍔 餐饮           ¥680 / ¥1000 │
│  ████████████░░░░░░  68%        │
│  剩余 ¥320 · 日均 ¥32           │
└─────────────────────────────────┘
```

- 分类图标 + 名称
- 进度条（绿/橙/红）
- 已花/预算/剩余/日均可用
- 点击可编辑该分类预算
- 超支时红色高亮 + 超支金额提示

### 4. 预算健康度评分

在总览卡片下方添加一行简洁的健康度提示：

| 状态 | 条件 | 显示 |
|------|------|------|
| 😊 健康 | 使用率 < 60% | "预算充裕，继续保持" |
| 😐 注意 | 60% ≤ 使用率 < 80% | "预算使用过半，注意控制" |
| 😰 紧张 | 80% ≤ 使用率 < 100% | "预算即将用完，请节制消费" |
| 🚨 超支 | 使用率 ≥ 100% | "已超出预算 ¥XXX" |

### 5. 快捷设置分类预算

当前只能设置总预算。优化为：
- FAB 点击弹出底部弹窗
- 可选择"总预算"或具体分类
- 分类列表展示当前是否已设预算
- 支持快速批量设置

### 6. 无预算时的引导优化

当前引导卡片太简单。优化为：

```
┌─────────────────────────────────┐
│  📊 开始管理你的预算              │
│                                  │
│  设置月度预算，帮你控制支出        │
│  养成良好的消费习惯               │
│                                  │
│  推荐预算：根据上月支出 ¥2,500    │
│  建议设置 ¥2,500 ~ ¥3,000       │
│                                  │
│      [ 一键设置推荐预算 ]         │
│      [ 自定义预算金额 ]           │
└─────────────────────────────────┘
```

- 根据上月实际支出推荐预算金额
- 一键设置按钮降低操作门槛

## 优先级建议

| 优先级 | 功能 | 工作量 | 价值 |
|--------|------|--------|------|
| P0 | 月份切换器 | 小 | 基础交互一致性 |
| P0 | 预算总览卡片（环形进度 + 核心数据） | 中 | 页面核心内容 |
| P1 | 分类预算卡片优化（日均可用、超支提示） | 小 | 数据洞察 |
| P1 | 预算健康度评分 | 小 | 情感化设计 |
| P2 | 无预算引导优化（推荐预算） | 中 | 转化率提升 |
| P2 | 快捷设置分类预算 | 中 | 操作效率 |

---

## 技术实现要点

- 环形进度图：复用报表的 Canvas 绘制能力
- 日均可用计算：`剩余金额 / (本月总天数 - 已过天数)`
- 月份切换：复用 `MonthPickerDialog` 公共组件
- 推荐预算：查询上月支出总额作为参考
- 所有卡片使用 `ElevatedCard`，和首页风格统一
